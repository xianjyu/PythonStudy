> ## python基础数据类型总览
+ 总共有下面数据类型
    - int       --->      整数类型
    - str       --->      字符串，可以保存少量的数据
    - bool      --->      布尔值，判断真假
    - list      --->      列表，可以存储大量的数据，用[]表示  
    - tuple     --->      元祖，是一种只读的列表，不发生改变，用()表示
    - dict      --->      字典，是Python中的映射性数据类型，由key-value键值对组成
    - set       --->      集合，保存大量的数据，不可重复
